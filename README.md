### `QQ`音乐新碟页面

- `ES6` /  `TypeScript`

> 简化版的`QQ`音乐新碟页面
>
> - 点击标签项动态切换下方列表内容
> - 鼠标悬停列表项会出现删除按钮，点击删除按钮可以将该条目删除
> - 移动优先，页面支持响应式，如列表项的列数，从1到5逐级过渡
> - 代码风格遵循 `standard.js` 规范
> - 使用动态创建 `DOM` 的方式来实现页面
> - 使用泛型方法请求 `json` 数据
> - 异步请求使用 `async/await`
